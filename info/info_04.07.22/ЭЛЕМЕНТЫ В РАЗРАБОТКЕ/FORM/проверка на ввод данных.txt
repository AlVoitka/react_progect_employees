1.Проверка припомощи цикла for:

Для начала, нужно поместить все в цикл, для того чтобы если условия не выполнятся, то вопросы будут задаваться по новой.
Условия из 3х аргументов:

- переменнай i просвоен 0, это начало цикла или с чего начинается цикл
- условие i < 1; , при котором цикл остановит свою работу ( те по условию отработает 1 раз)
- i++  это шаг цикла ( то что будет происходить каждый раз в новом цикле) 
И  цикл продолжается  до выполненипя условия i<1 

for (let i = 0; i < 1; i++) {

    const ask_1 = prompt('Are you a cargo owner or logistician?','').trim();
    const ask_2 = prompt('Do you want a consultation?','').trim();

    if(ask_1 != null && ask_2 != null && ask_1 !='' && ask_2 !='' && ask_1.length < 11 && ask_2.length < 11 ) {
        console.log('Done!');
        userDB.answers[ask_1] = ask_2;
    } else {
        console.log('Error!');
        i--;
    }
}

Если вопрос не null, не пустая строка '', длина символов не более 11, то пыпланяем условия {}
Иначе else {
        console.log('Error!');  - в консоль ошибка
        i--;  ---- и цикл возврыщает назад на вопросы.
    }

метот .trim() - проаерит не ввел ли пользовател пустой пробел и если ввел, то удалит его и снова вернет пустую строку.

Цикл будет выполняться пока одно из условий будет правдивым.

Создаю функцию:

function valid () {

    userDB.answers = ask_1;

    while (ask_1 == '' || ask_1 == null || isNaN(ask_1)) {
