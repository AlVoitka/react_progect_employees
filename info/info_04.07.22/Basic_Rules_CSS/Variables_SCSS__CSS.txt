/////////// Синтаксис CSS  /////////////////    

1. Локальное объявление переменной

.wrapper {
        --text-color: red;  --- Объявил переменную и она существует только на сомом блоке  или на элементах внутри wrapper (область видимости)
}

.contact {
        color: ver(--text-color);  -- применил её
}

2.
Глобальное объявление переменной

Объявляется внутри псевдокласса :root

:root {
        --text-color: red;
        --big-text: 30px;
}

и теперь она глобаляная для всего документа

.descr {
        font-size: var(--big-text);  -- вот использовал
}

.descr {
        font-size: var(--big-text, red);  -- можно и так, var() выступает как функция, и использую ее как колл-бек функцию
                                          тоесь если не выполняется первое условие --big-text( к примеру код ее не видит),
                                         то выполняется второе - red.
}


////////////         Основное отличие:  //////////////////

Переменные CSS находятся в DOM-дереве и сними можно взаимодействовать:
1 частности при адаптации,
2 в скриптах
а SCSS-переменные -толко в файлах .csss и потом компелируются в css где уже преобразованы в обычный код css.
Но css переменные можно легко использовать в scss-файлах!!

1 Пример при адаптации

.wrapper {
        --text-color: red; 
        --big-text: 30px; 
}
@media (max-width: 600px) {
        :root {
        --text-color: blue;  -- вот в медиаправилах поменял цвет при малом экране
        --big-text: 90px;  -- и размер
        }
}

2 Пример в скрипте
Если нужно по одному клику изменить туму на сайте или какието элементы

const root = document.querySelector(':root'),   -- получаю корневой элемент 
      rootStyles = getComputedStyle(root);  --- а также все стили(значения) в нем лежащие
      btn = document.querySelector('button');  --- кнопка

  btn.addEventListener('click', ()=> {
    root.style.setProperty('--text-color', 'blue');
  })

  при клике на кнопку меняется цвет
