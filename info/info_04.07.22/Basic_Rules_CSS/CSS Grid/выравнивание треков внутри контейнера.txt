выравнивание треков внутри грид-контейнера  при помощи свойства aling-self и justify-self

https://cssreference.io/property/align-self/  ---тут визуалка


1.
align-self: stretch;  -- это значение по умалчанию, те растягивает трек на всю доступную высоту/ по вертикальной оси

justify-self: start;  -- это выравнивание по горизонтальной оси

и это свойство нужно задавать каждому треку:

#amsterdam {
    grid-column-start: 4;
    grid-row-start: 1;
    grid-row-end: 6;
    align-self: center;  --- вот свойство для отдельного трека
}


2. 
А для того чтобы применить похожее свойство, но для всех и каждого трека использую свойство aling-item
и задается оно для самого контейнера с элементами

.grid-container {
    align-items: start;  -- ровняет по вертикали
    justify-items: start; -- по горизонтали
}

3.
также можно использовать для контейнера внутри которого треки свойства 

.grid-container {
    justify-content - выравняет по горизонтали
    aling-content - по вертикали
}
