

massive [] - это часный случай объекта и служит для расположения данных строго по порядку.
массив это перечень информации по порядку, каждая сущность имеет строгий номер по порядку.
Порядок имеет коючевую роль.

let arr = ['plum.png', 'orange.jpg', 'apple.bmp', 999];
console.log(arr[0]); // выводит plum.png
console.log(arr[1]); // orange
console.log(arr[2]); // apple
console.log(arr[3]); // 999
