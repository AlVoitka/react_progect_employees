 вешаю обрабочтик на весь индекс html на событие 'keydown' 

 используя объект события и свойство -code (опредеяет код клавиши), если оно Esc то закрываем окно.


document.addEventListener('keydown', (e) => {
        if (e.code === "Escape" && modal.classList.contains('show')) { 
            closeModal();   
        }
    });


    Но для того чтобы действие по закрытию окна происходило только во время открытого окна, в условии прописываю:

    && modal.classList.contains('show')  --- тут я проверяю содержится ли клас 'show' у модалки.Те показано ли оно!
