Метод push добавляет неограниченное количество элементов в конец массива. При этом исходный массив изменяется, 
а результатом возвращается новая длина массива.

Синтаксис
массив.push(элемент, элемент, элемент...)

Пример 
Давайте в конец массива добавим еще два новых элемента:

let arr = ['a', 'b', 'c', 'd', 'e'];

arr.push('1', '2');
console.log(arr);
Результат выполнения кода:

['a', 'b', 'c', 'd', 'e', '1', '2']
