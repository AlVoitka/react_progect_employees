Метод forEach позволяет последовательно перебрать все элементы массива.
Это метод никогда не возвращает новый массив, он просто его переберает и назначает какое то действие на каждый из элементов.


Метод в параметре получает функцию, которая выполнится для каждого элемента массива. 
В эту функцию можно передавать 3 параметра.

Если эти параметры есть (они не обязательны), то в первый автоматически попадет элемент массива,
 во второй попадет его номер в массиве (индекс), а в третий - сам массив.


1. в консоли нахожу нужный элемент для удаления и смотрю как лучше его получить ( всегда лучше искать селектор)
2. Получаю элемент:

const elem = document.querySelectorAll('.promo_adv')

таких элементов несколько, поэтому я получаю html колллекцию, а это объект.

3. Раз уж там несколько элементоа, то нужно их перебрать:

elem.forEach(item => {  --- каждый элемент по порядку ( тут применена стрелочная функция)
    item.remove();
})

ну и к примеру с обычной функцией:

elem.forEach(function(item) {  --- каждый элемент по порядку ( тут применена обычная функция)
    item.remove();
})
