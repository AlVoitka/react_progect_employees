.some()
Берет масииив, проверяет его и если хотябы один элемент подгодит по условию, то метод вернет - true
Метод some() проверяет, удовлетворяет ли какой-либо элемент массива условию, заданному в передаваемой функции.
Обратите внимание: метод возвращает false при любом условии для пустого массива.

имею массив 

const some = [4, 'vsevw', 'vvvvvvvvvf']

сосдам функцию по проверке есть ли в массиве число:

some.some(item => typeof(item) === 'number'); // true   --- есть одно число







.every()
толке нсли все элементу подходят под условие, тогда метод вернёт - True
Метод every() проверяет, удовлетворяют ли все элементы массива условию, заданному в передаваемой функции.
Обратите внимание: метод возвращает true при любом условии для пустого массива.

имею массив 

const some = [4, 'vsevw', 'vvvvvvvvvf']

сосдам функцию по проверке все ли тут числа:

some.every(item => typeof(item) === 'number'); // false - тк не все элементы являются числои
