
Зачастую нам надо повторять одно и то же действие во многих частях программы.
Например, необходимо красиво вывести сообщение при приветствии посетителя, при выходе посетителя с сайта, ещё где-нибудь.
Чтобы не повторять один и тот же код во многих местах, придуманы функции. Функции являются основными «строительными блоками» программы.

Основными моментами является объявление и вызов функции.
Внегласное правило - называть функцию нужно глаголом, тк это какое то действие




1.

let n1 = 3;
let n2 = 5;

function num() {   ---- вот я объявил функцию
    let result = n1 + n2;
    console.log(result); // 8
}
num();   --- вот я ее вызвал

Переменные объявленные внутри функции снаружи недоступны - и называется локальной.
А вот переменные объявленные вне ( снаружи) функции называются глобальными и они доступны внутри функции.







2. Аргументы

function sayHi (text) {
    console.log(text); // Hi
}

sayHi('Hi');

была просто функция без аргумена, я ее вызвал sayHi() и задал аргумент - строку 'Hi', sayHi('Hi').
теперь в функции я принемаю этот аргумент function sayHi (text) {}, и внутри функции с ним работаю ( использую его).




/////////////////// Отличный пример /////////////////////////

function convertBYN(value) {
        console.log(2.55*value);
    };
convertBYN(100);

Функция по перещету курса валюты. Задаю любое значение в convertBYN(100);, а функция принимает его как аргумент 
и пресчитывает. Те изначально функция не знает какое значение будет задано и это важная деталь.


А еще можно добавит так:

let currUSD = 2.65;  --- к примеру это значение динамически приходит от банка

function convertBYN(value, curr) {
        console.log(curr * value);
    };
convertBYN(100, currUSD);  --- передаю 2 аргумента. Функция принемает их и сними работает. Она универсальна и не зависит
 от фиксированных значений.






пример с двумя аргуметами:

function mySing(artist, track) {     ---- тут принял эти аргументы
        let sing = artist + ' поёт ' + track;  ----  вот их использовал
        console.log(sing);// витя поёт как конь
    }

mySing('витя', 'как конь'); --- вот вызвал функцию и передал 2 аргумента





пример с двумя аргуметами:

function mySing(artist, track) {
    return artist + ' поёт ' + track;    ---- использую конкатенацию
}

let result = mySing('Толя','Централ');
console.log(result);// Толя поёт Централ

Переменная result - в нее записана и соттветсвенно вызвана функция ySing с двумя аргументами.
