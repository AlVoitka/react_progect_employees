  
  
  Циклы ( 3 вида)
  
1. while  - В переводе покуда, до тех пор пока  какие то условия выполняются, то совершаются действия
  
let num = 50;

 while (num < 55) {  --- пака num меньше 55, то выполняется действие
   console.log(num);  ---- вот действие и оно бесконечно, тк num 50 всегда меньше 55
  но стоит добавить num++ и код выполнит действия от 50 каждый раз прибавит 1 пока не дойдет до 54
   
while (num < 55) { 
  console.log(num);  ///// 50  51  52  53  54  
  num++;

}


2. do - сделай, сначала цикл делает, а потом проверяет условия

let num = 50;

do {
   console.log(num); //// 50 51  52  53  54 
   num++;
}
while (num < 55);




3.  for -



for (let i = 1; i < 8; i++) {
    console.log (i); // 1  2  3  4  5  6  7  
}

Условия из 3х аргументов:

- переменнай I просвоена 1, это начало цикла или с чего начинается цикл
- условие i < 8; , при котором цикл остановит свою работу
- i++  это шаг цикла ( то что будет происходить каждый раз в новом цикле) 
И  цикл продолжается  до выполненипя условия i>8 тоесть в каждом повторении цикл увеличивается на 1
