
Используйте jQuery в качестве модуля узла
Если вы хотите использовать jQuery в качестве модуля узла и связать его, вам следует установить jquery из npm

npm install jquery


Затем импортируйте его в свой код


import $ from "jquery"; -- один из вариантов синтаксиса
import abra from "~abra/abra.js"

import jquery from './jquery';  ---- вот так
