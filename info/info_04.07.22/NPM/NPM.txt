Для того чтобы инициализировать в проект NPM нужно для начала  скачать и установить NODE.js
Это стнова для работы с npm



1.
Всегда перед тем как устанавливать дополнительные зависимости npm в проект, нужно создать npm проект:

npm init

команда запускает процесс создания файла package.json
полследуе несколько вопросов на которые можно ответить

Как итог появляется файл: package.json  ---!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
В нем будет вся информация о проекте, а также он содержит в себе инфу о npm-пакетах которые будут использованы в проекте.
Тоесть теперь устанавливая пакет в проекте, он автоматически  добавляется в файл package.json

2.
Теперь можно устанавливать пакеты:

к примеру установлю пакет json-server

Есть такой момент как устанавливать пакет глобально или лакально. 
Если глобально - то он работае везде по системе, в любом проекте и папке.
Если лакально ( предпочтительно!!!) - то в package.json запишется инфа о нем и его версия,а это важно тк в любом месте любой пользователь
будет четко видеть какие пакеты использовались в проекте и что нужно установить. 


в терминале проекта:

npm i json-server    --- так локально
npm i json-server -g   --- так глобально

можно еще добавлять уточняющий  флаг:

npm i json-server --save-dev         --- так для разработки
npm i json-server --save         --- так для использования   (лучше с флагом стили будут использоваться в самом приложении а не толькл для разработки)

как итог:
- папка node_modules !!!!!!!!!!!!!!  --- внутри ее вообще ничего не трогатЬ!!!  также обязательно ее в . GITignore - на очень тяжелая
- файл package-lock.json
- в package.json добавилась инфа об этом:

        "devDependencies": {   ---- тут dev если команда была npm i json-server --save-dev 
            "json-server": "^0.16.1",


Представим что я склонировал чужой проект из гита и внем естественно нет папки node_modules.
Но зато есть файл package.json где имеется вся информация о зависимостяхю. 
В терминале ввожу  npm i (npm install)  
и все пакеты устанавливаются соответственно инфе из  package.json   


2.
Для того чтобы обновить все пакеты что в прописаны в package.json

npm update --save
